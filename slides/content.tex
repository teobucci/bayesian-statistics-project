The following Python code adds two numbers and display the result using \verb|print()| function:
\rule{\textwidth}{1pt}
\scriptsize
\begin{minted}{python}
    # This program adds two numbers
    num1 = 1.5
    num2 = 6.3
    # Add two numbers
    sum = num1 + num2
    # Display the sum
    print('The sum of {0} and {1} is {2}'.format(num1, num2, sum))
\end{minted}
